Statlas

Statlas is a tool to enable laypeople to easily visualize mappings from a set of values to a set of regions in a way that is interesting, useful and beautiful.

The current version (blog post) is very much in beta, but wholly functional for the regions currently provided.

Feedback

This is an beta release geared towards the Netherlands, but Statlas is intended to be a general mapping tool. We are working towards adding more regions for any geographical/administrative definition you can imagine. You can help us by voicing your demands and pointing us towards appropriately licensed geometry (in Shapefiles or other).

Any and all feedback can be directed here until more specific channels are available.

Credits

Statlas is created by Monster Swell in collaboration with Fluxility, and Alexander Zeh with funding from the Dutch Fund for Press Innovation. The cartography is powered by Polymaps and thumbnails are generously provided by url2png.

The source code is available at Github.
